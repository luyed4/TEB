Curso de Operador de Micros

Descripcion

En el mundo de la Ofimática se creado un conjunto de aplicaciones para facilitar al usuario las tareas como hacer un documento, una presentación, una hoja de cálculo, imprimir un archivo, etc.

Conocimientos adquiridos

Todos los principios básicos para manejar, conocer e investigar acerca de las aplicaciones de ofimática para realizar una tarea en particular en casa, oficina o departamento.

Tecnologías

  • Debian GNU/Linux
  • Aplicaciones de Ofimática
